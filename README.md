## 🚀 À propos de moi

Développeur **Full Stack** passionné par la conception d’applications web et mobiles modernes.  
Je travaille principalement dans l’écosystème **JavaScript / TypeScript**, avec une attention particulière portée à la **qualité du code**, à la **maintenabilité** et à l’**expérience utilisateur**.

- 🔭 Je développe des projets full stack autour de **React, Node.js, Express et MySQL**
- 🌱 Je consolide mes compétences en **TypeScript**, **architecture backend** et **applications scalables**
- 📱 J’explore aussi le développement mobile avec **React Native / Expo**
- 💼 Je recherche une **alternance / opportunité professionnelle** en développement web
- 🎯 Mon objectif : construire des produits utiles, performants et bien structurés

## 💻 Projets mis en avant

### ⚡ MoveUp
Plateforme **fitness full stack** dédiée à la planification d’entraînements, au suivi utilisateur et à l’interaction avec des coachs.

- **Frontend :** React 19, Vite, React Router 7, FullCalendar
- **Backend :** Node.js, Express, MySQL
- **Fonctionnalités :** dashboard personnalisé, catalogue d’exercices, favoris, gestion de profil, authentification classique + Google OAuth
- **Qualité :** TypeScript, Biome, organisation en **monorepo**, workflow structuré

🔗 **Frontend :** [MoveUp](https://github.com/DevNexus59/MoveUp)  
🔗 **Backend :** [API-MoveUp](https://github.com/DevNexus59/API-MoveUp)

---

### 🧩 Pickit
Application web **peer-to-peer** de prêt de matériel sportif, pensée autour du partage, de l’accessibilité et de la consommation responsable.

- **Tech :** React, TypeScript, Vite, Node.js, Express, SQL
- **Fonctionnalités :** authentification, gestion d’annonces, filtres de recherche, emprunt avec caution, chat temps réel, wishlist, dashboard admin
- **Contexte :** projet d’équipe mené en méthodologie **Scrum** avec **Jira**

🔗 **Repo :** [Pickit](https://github.com/DevNexus59/Pickit)

---

### ☕ CoffeeGotchi
Mini-jeu mobile inspiré du principe **Tamagotchi**, développé avec l’écosystème **React Native / Expo**.

- **Tech :** Expo, React Native, TypeScript, React Navigation
- **Objectif :** explorer le développement mobile et la logique de jeu interactive

🔗 **Repo :** [CoffeeGotchi](https://github.com/DevNexus59/CoffeeGotchi)
